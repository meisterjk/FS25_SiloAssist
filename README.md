# FS25 Silo Assist

**Author:** mj  
**Game:** Farming Simulator 25

## 🇩🇪 Deutsch

Automatische Höhensteuerung für Schiebeschild und Schaufel im Bunkersilo.

### Funktionen

- **Durchfahrts-Modus**: Schild wird automatisch auf Silage-Höhe + Offset gehalten, mit ein- und ausfahrenden Rampen
- **Keilsilo-Modus**: Schild fährt Keil-Steigungen, die mit jedem Durchlauf höher werden
- **Automatische Werkzeug-Erkennung**: Erkennt Schiebeschild (3-Punkt, Radlader, Frontlader) oder Schaufel
- **Schaufel-Modus**: Kontinuierliches Auskippen beim Rückwärtsfahren
- **Pre-Positioning**: Schild senkt sich bereits vor dem Silo auf Arbeitshöhe ab
- **Festfahren-Erkennung**: Wenn das Fahrzeug feststeckt, wird das Schild automatisch angehoben
- **Neigungskontrolle**: Kompensiert Fahrzeug-Pitch, Schild bleibt im richtigen Winkel
- **HUD**: Kompaktes Info-Display mit Füllstand, Verdichtung, Soll/Ist, Modus-Wahl, Offset-Einstellung
- **Toggle-Switch**: Assist direkt im HUD an/aus schalten (oder per Tastatur Strg+L)

### Tastenkombinationen

| Taste | Funktion |
|-------|----------|
| Strg+L | Assist an/aus |
| RStrg+L | Modus wechseln (Durchfahrt/Keil) |
| Strg+H | HUD ein/aus |

### Installation

ZIP-Datei in den Mod-Ordner kopieren. Im Spiel Konsole öffnen und `reloadSiloAssist` eingeben zum Neuladen während der Entwicklung.

## 🇬🇧 English

Automatic height control for push blades and shovels in bunker silos.

### Features

- **Drive-through mode**: Blade is held at silage height + offset, with entry/exit ramps
- **Wedge mode**: Blade follows wedge profiles that increase with each pass
- **Auto tool detection**: Recognizes leveler (3-point, wheel loader, front loader) or shovel
- **Shovel mode**: Continuous dumping while reversing
- **Pre-positioning**: Blade lowers to working height before entering the silo
- **Stuck detection**: Automatically raises the blade when the vehicle gets stuck
- **Tilt control**: Compensates vehicle pitch, blade stays at the correct angle
- **HUD**: Compact info display with fill level, compaction, target/actual, mode selection, offset setting
- **Toggle switch**: Turn assist on/off directly in the HUD (or via keyboard Ctrl+L)

### Controls

| Key | Function |
|-----|----------|
| Ctrl+L | Toggle assist on/off |
| RCrtl+L | Cycle mode (drive-through/wedge) |
| Ctrl+H | Toggle HUD |

### Installation

Copy ZIP file to your mods folder. Open the console in-game and type `reloadSiloAssist` to reload during development.
